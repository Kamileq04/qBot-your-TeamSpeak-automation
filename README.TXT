Requirements/Wymagania
 1. php 7.2
 2. php7.2-cli
 3. php7.2-curl
 4. screen

 Intalation/Instalacja
 1. wget https://github.com/stalkerlifehack/qBot-your-TeamSpeak-automation/archive/master.zip
 2. unzip master.zip
 3. mv qBot-your-TeamSpeak-automation-master qBot_v4.1
 4. cd qBot_v4.1
 5. chmod 0775 run 
 6. cp config.php.SAMPLE_EN config.php /  cp config.php.SAMPLE_PL config.php

Commands/Komendy
    ./run start
    ./run stop
    ./run restart
    ./run daemon
